🚀 LS 标注助手

在浏览器和桌面之间桥接 Label Studio 标注操作，一键填充、提交、跳转。

🌐 方式一：Web UI 推荐

桌面工具启动后，浏览器会自动打开 Web 管理界面，操作更直观：

  • 🎨 现代界面设计，明暗主题
  • 📋 模板拖拽管理，一键执行
  • 📡 通过 API 自动查找下一任务（LS 版本升级也不受影响）
  • ⚡ 无需安装任何软件，浏览器即可操作

👉 启动桌面工具后，访问：
http://127.0.0.1:17892/web/

🔖 方式二：安装 LS连接器 必要

无论使用 Web UI 还是旧版桌面工具，都必须安装此书签，用于在 LS 页面注入桥接脚本。

第一步：确认桌面工具已启动（桥接服务器运行在 port 17892）

第二步：把下面这个按钮拖到浏览器收藏栏：

🔌 LS连接器

第三步：打开 Label Studio 具体任务页，点击收藏栏的 LS连接器。

第四步（可选）：回到 Web UI 页面，状态栏显示「已连接」即可操作。

🔄 使用流程

  1. 启动桌面工具（自动打开 Web UI）
  2. 在 LS 标注页点击书签栏「LS连接器」
  3. 在 Web UI 中选择备注模板点击执行
  4. 工具自动完成：选择不符合 → 填备注 → Add → Submit → 跳转下一任务

📖 浏览器兼容说明

Safari

  1. 菜单栏点击：显示 → 显示个人收藏栏。
  2. 用鼠标按住「LS连接器」拖到个人收藏栏。

Chrome / Edge

  1. 确保书签栏可见（Ctrl+Shift+B / Cmd+Shift+B）。
  2. 将「LS连接器」拖到书签栏。

如果拖不上去

  1. 新建任意书签。
  2. 名称设为 LS连接器。
  3. URL 替换为项目 bridge/bookmarklet.txt 中的整段 javascript: 代码。
