# 🚀 LS 标注助手 — 快速上手指南

## 对首次使用的同事

### 第一步：下载并解压

1. 拿到 `LS标注助手-Windows-x64.zip`
2. 解压到任意文件夹（建议在桌面或 D 盘）
3. 解压后你会看到这些文件：
   ```
   📁 LS标注助手/
   ├── 🏃 LS标注助手.exe      ← 主程序（双击运行）
   ├── 📖 install_bookmarklet.html  ← 书签安装页面
   └── 📖 README.md
   ```

### 第二步：安装浏览器书签

1. 双击打开 `install_bookmarklet.html`（浏览器自动打开）
2. 把页面里的 **🔌 LS连接器** 按钮拖到浏览器的**书签栏**
   - 💡 *如果书签栏没显示：Chrome 按 `Ctrl+Shift+B` 显示书签栏*

### 第三步：启动程序

1. **双击 `LS标注助手.exe`** 运行
2. 浏览器会自动打开 Web UI 界面（`http://127.0.0.1:17892/web/`）
3. 首次启动可能需要同意防火墙，点击「允许」

### 第四步：连接 Label Studio

1. 浏览器中打开你的 Label Studio 标注页面
2. **点击浏览器书签栏的「LS连接器」**
3. 页面右上角会闪现「LS Bridge 已连接」提示
4. 回到 Web UI 页面，顶部状态栏会显示「已连接」

### 第五步：开始使用

1. 在 Web UI 中可以直接点击备注模板按钮（如「说话内容有误」）
2. 工具会自动完成：选择不符合 → 填备注 → 提交 → 跳到下一任务
3. 重复操作即可，全程不需要动鼠标

---

## 常见问题

### ❓ 双击 `.exe` 没反应
- 右键 `LS标注助手.exe` → **以管理员身份运行**
- 或者安装微软运行库：https://aka.ms/vcredist

### ❓ 提示「Windows 保护了你的电脑」
- 点击「更多信息」→「仍要运行」
- 这是开源工具，没有病毒

### ❓ 浏览器不自动打开
- 手动在浏览器地址栏输入：`http://127.0.0.1:17892/web/`

---

## 注意事项

⚠️ 每次使用前都要先启动 `LS标注助手.exe`
⚠️ 关闭程序：在命令行窗口按 `Ctrl+C` 或直接关掉窗口
⚠️ 不要关闭或刷新 Web UI 页面 — 它只是控制面板
